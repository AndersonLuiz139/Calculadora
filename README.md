# 🧮 Calculadora Web

Uma calculadora web moderna desenvolvida com **HTML, CSS e JavaScript puro**, focada em usabilidade, design e boas práticas de desenvolvimento front-end.

---

## ✨ Funcionalidades

- Operações matemáticas completas (+, −, ×, ÷)
- Suporte a teclado
- Histórico de cálculos
- Persistência de dados com **localStorage**
- Alternância entre tema claro e escuro 🌙☀️
- Interface moderna (glassmorphism)
- Layout totalmente responsivo

---

## 🧠 Aprendizados

Durante o desenvolvimento, foi possível praticar:

- Manipulação de DOM com JavaScript
- Organização de código utilizando classes
- Gerenciamento de estado no front-end
- Uso de localStorage para persistência
- Boas práticas de UI/UX
- Estruturação de commits

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📱 Responsividade

A aplicação foi desenvolvida para funcionar em diferentes tamanhos de tela:

- Desktop 💻
- Tablet 📱
- Mobile 📱
